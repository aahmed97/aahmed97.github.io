Projects

Linear SDN

Written in C, this project simulates a linear SDN where clients can communicate with one another using named pipes. The clients communicate with the server hosted on the local device using sockets. The sever handles query requests from up to 7 clients, directing packets to the correct destination and dropping those with an invalid destination IP address.

CardioBook

An android application which helps to track health information such as heart rate, systolic pressure, diastolic pressure, and the dates where the measurement was taken.

*Currently a work in progress*

Mountain Car

Written in Python, this project implements a the reinforcement learning algorithm SARSA using tile coding and replacing trace to implement a solution for the mountain car problem. The "car" can not produce enough throttle to escape it's position in between the two mountains. The agent learns that a back and forth motion utilizing momentum and gravity is the most effective method of escape.
